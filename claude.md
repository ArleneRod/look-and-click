# Look and Click - Project Documentation

## Overview
A modern React application built with Vite featuring bilingual support (Spanish/English) and interactive UI components.

## Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Language**: JavaScript/TypeScript
- **UI Libraries**: Framer Motion, AOS
- **Icons**: React Icons
- **Forms**: EmailJS

## Development

### Local Development
```bash
npm install
npm run dev
```

### Build
```bash
npm run build
```

### Preview Production Build
```bash
npm run preview
```

## Deployment

This project is automatically deployed to GitHub Pages on every push to the `main` branch.

**Live URL**: https://arlenerod.github.io/look-and-click

### Deployment Process
- GitHub Actions automatically builds and deploys on push
- No manual deployment needed
- Build artifacts are deployed to the `gh-pages` branch
